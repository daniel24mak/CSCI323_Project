## Running the Pipeline

### Full demo (recommended)

```bash

```

```bash
python demo.py --csv path/to/your/data.csv
```

## Pricing Engine

The pricing formula is:

```
recommended_premium = predicted_expense
                    Г— BASE_LOADING_RATE (10%)
                    Г— risk_multiplier
                    Г— (1 в€’ discount)
```

| Risk Tier | Multiplier |
|-----------|-----------|
| LOW_RISK | 0.85 |
| MEDIUM_RISK | 1.00 |
| HIGH_RISK | 1.25 |

Customers with `discount_eligibility = yes` receive an additional **10% discount**.

## Dataset

`data/raw/medical_insurance.csv` вЂ” 1 338 records with columns:

| Column | Type | Description |
|--------|------|-------------|
| age | int | Customer age |
| gender | str | male / female |
| bmi | float | Body Mass Index |
| children | int | Number of dependants |
| discount_eligibility | str | yes / no |
| region | str | southwest / southeast / northwest / northeast |
| expenses | float | Actual medical expenses (regression target) |
| premium | float | Existing premium (excluded from features) |
